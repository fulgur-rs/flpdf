# Golden Compatibility Matrix

Matrix of where flpdf and qpdf agree under at least one comparator strategy
on the curated corpus. Generated by the compat_matrix_baseline integration
test.

## Comparator strategies

- **byte-equal**: byte-for-byte identical output
- **qpdf-json**: qpdf --json=2 representations match (requires qpdf at runtime)
- **structural**: flpdf-parsed object graphs match (encoding-only stream
  dict keys excluded when decoded content matches)

The **flpdf-sha** column is a stable 64-bit FNV-1a fingerprint of flpdf's
output bytes. It changes whenever flpdf's output changes, so the baseline
detects byte-level drift even when every comparator verdict stays at
`diverge`.

## Review cadence

Re-bless this file when:
- qpdf binary is upgraded (qpdf-json comparator may shift)
- flpdf changes byte / structural output for any covered fixture × flag
- New fixtures or flags are added to the corpus

Update via: `BLESS=1 cargo test --test compat_matrix_baseline`
