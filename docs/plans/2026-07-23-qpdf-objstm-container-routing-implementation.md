# qpdf ObjStm Container Routing Parity Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make linearized Generate and Preserve modes classify each ObjStm container once, retain that qpdf-compatible route through placement, and eliminate member-based reclassification.

**Architecture:** Introduce an internal `RoutedObjStmBatch` that keeps members and their canonical `ContainerPart` together. Generate mode retains the existing union route; Preserve mode classifies each surviving source container as a unit; the writer derives anchors from the retained route instead of recreating the part from classic partitions.

**Tech Stack:** Rust workspace, qpdf 11.9.0 oracle, Cargo integration tests, committed PDF compatibility fixtures and goldens, `qpdf-zlib-compat`.

## Global Constraints

- qpdf 11.9.0 source and observed output are the behavioral oracle.
- Existing byte-identical Generate output must not change.
- Preserve mode must keep surviving source-container membership and classify the container union.
- Write a failing test and observe the intended failure before production changes.
- Do not change non-linearized object-stream behavior.
- Final committed-HEAD patch coverage must be 100%.
- Run `cargo fmt --all -- --check`, workspace all-feature clippy, crate tests, and workspace tests before completion.
- Close `flpdf-5mv0`, push Beads state, commit all code, and push the git branch.

let assigned = self.renumber_assigned_refs();

    crate::pages::page_refs(pdf)?.into_iter().collect();

let mut containers: Vec<Vec<ObjectRef>> = Vec::new();

for (_container_num, mut members) in groups {
    members.sort_by_key(|(idx, _)| *idx);
    let mut surviving = Vec::new();
    for (_idx, obj_ref) in members {
        if page_dicts.contains(&obj_ref) || Some(obj_ref) == catalog {
            continue;
        }
        if length_exclusions.contains(&obj_ref) || !assigned.contains(&obj_ref) {
            continue;

        if is_eligible_for_objstm(obj_ref, obj, ctx) {
            surviving.push(obj_ref);
        }
    }
    if !surviving.is_empty() {

    }
}

- [ ] **Step 5: Route preserved containers by their qpdf union**

Compute routes once:

```rust
let routes = route_objstm_containers(pdf, &containers)?;
let mut open_document_batches = Vec::new();
let mut part3_batches = Vec::new();
let mut part4_private = Vec::new();
let mut part4_shared = Vec::new();
let mut part4_rest = Vec::new();

for (members, route) in containers.into_iter().zip(routes) {
    match route {
        ContainerPart::OpenDocument => open_document_batches.push(members),
        ContainerPart::FirstPage => part3_batches.push(members),
        ContainerPart::OtherPagePrivate => part4_private.push(RoutedObjStmBatch {
            members,
            route,
        }),
        ContainerPart::OtherPageShared => part4_shared.push(RoutedObjStmBatch {
            members,
            route,
        }),
        ContainerPart::Rest => part4_rest.push(RoutedObjStmBatch {
            members,
            route,
        }),
    }
}
let mut part4_batches = part4_private;
part4_batches.extend(part4_shared);
part4_batches.extend(part4_rest);
```

Return all three batch groups. This deliberately removes Preserve's previous
individual owner re-chunking and its special `/Pages`/`/Info` promotion.
